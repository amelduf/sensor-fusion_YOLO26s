# sensor-fusion_YOLO26s: Comparative Study of Camera-Radar Fusion Methods for Vehicle Detection Using YOLO26s

This project contains the code of a construction of a dataset on CARLA simulator, the train of 4 different Camera-Radar Fusion Methods for Vehicle Detection Using YOLO26s (camera only baseline, SAF-yolo26s, early fusion and Multi-scale fusion) and the application of the SAF-yolo26s model on CARLA simulator. The demo of the implementation of SAF-YOLO26S on CARLA os available on: https://youtu.be/0mlI5mkCTl0

In the study, to construct the dataset based on nuScenes, we use the code providede by the SAF-FCOS repository (https://github.com/Singingkettle/SAF-FCOS/tree/master#saf-fcos-spatial-attention-fusion-for-obstacle-detection-using-mmwave-radar-and-vision-sensor).
The full paper of the SAF-FCOS study, is available at: https://www.mdpi.com/1424-8220/20/4/956.
